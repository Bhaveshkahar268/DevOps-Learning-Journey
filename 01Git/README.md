# 📖 Introduction

Git is a **Distributed Version Control System (DVCS)** developed by **Linus Torvalds** in 2005. It helps developers track changes, collaborate efficiently, and maintain the complete history of a project.

GitHub is a cloud-based platform that hosts Git repositories, enabling developers to collaborate, review code, and manage software projects.

# 💡 Best Practices

- ✅ Commit frequently
- ✅ Write meaningful commit messages
- ✅ Pull latest changes before pushing
- ✅ Create a new branch for every feature
- ✅ Never commit passwords or API keys
- ✅ Use `.gitignore` for unnecessary files
- ✅ Review changes using `git diff`
- ✅ Keep commit history clean
